# 🤖 AI 应用技术汇总

> 收集整理 AI 应用开发相关的技术文档、框架、工具和最佳实践

## 📂 项目结构

```
ai-app-tech-hub/
├── docs/                   # 技术文档
│   ├── llm/               # 大语言模型相关
│   ├── rag/               # RAG（检索增强生成）
│   ├── agent/             # AI Agent 技术
│   ├── prompt-engineering/# 提示词工程
│   └── mcp/               # Model Context Protocol
├── frameworks/            # AI 应用框架
├── tools/                 # 工具与平台
├── best-practices/        # 最佳实践
└── resources/             # 学习资源
```

## 🔥 热门技术方向

| 方向 | 说明 |
|------|------|
| LLM 应用开发 | 基于 GPT、Claude、Gemini 等大模型的应用开发 |
| RAG | 检索增强生成，结合企业知识库构建智能应用 |
| AI Agent | 自主决策与多步骤任务执行的智能体 |
| MCP | Model Context Protocol，模型上下文协议 |
| Prompt Engineering | 提示词设计与优化技术 |
